Procedure
  • 3 bit Binary Wighted Resistors
    1. Connect the components as mentioned below:
      L1-L5, L4-L6, L3-L5.(For eg. click on 1 and then drag to 5 and so on.)
    2. Click on 'Check Connection' button to check the connections.
    3. If connected wrong click on 'Delete all connection' button to erase all the connections.
    4. Set the Resistance (R) value (1 Kohm - 10 Kohm).
    5. Set the Feedback Resistance (Rf) value same as the Resistance (R) .
    6. Set input voltage (Vin).
    7. Intially B2B1B0are set to 000.(B2B1B0 are 3 bit input)
    8. Now note the output voltage and click on 'Add to table' button.
    9. Change the B2B1B0 and note the output voltage and click on 'Add to table' button.
    10. Repeat the experiment for another set of resistance value.
